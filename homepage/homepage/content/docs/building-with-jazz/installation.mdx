# Installation

Adding Jazz to an existing application is quick and easy. In just a few minutes, you can integrate Jazz's real-time collaboration features into your project.

If you're starting a new project, check out our [`create-jazz-app`](/docs/tools/create-jazz-app) tool for a streamlined setup experience.

Integrating Jazz is straightforward. You just need to install the packages and you're ready to go.

## Install dependencies

First, install the required packages:

<CodeGroup>
```bash
pnpm install jazz-react jazz-tools
```
</CodeGroup>

## Next Steps

Now that you've installed Jazz, the next step is to define your data structures with [Schemas](/docs/building-with-jazz/schemas).
